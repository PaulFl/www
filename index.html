Capacitive sensor for lamp control (using Raspberry Pi)

Demo: YouTube video

This is my first project about home automation. It allows you to control one lamps through a relay (always be very carefull with high voltage ;)) just by touching a can. In this project, I'm using a Raspberry Pi but it can also be done with an Arduino or any microcontroller! I've added a touch feedback in the can with a small LED which turns off when you're touching the can. A Python script running as a deamon on the RPi does the logic.

I'm using a 1mâ„¦ resistor (one leg is connected to send pin and the other to receive pin). You then just have to conect the can to the receive pin. You may have to adjust the threshold value into the script

Electric Longboard

Prototype version alpha: YouTube video

  • Range: ~10km

  • Bluetooth speaker (hasn't been working for long)

  • custom home-made nunchuck remote
  • USB Hub
  • indication LEDs
  • MCU to receive/modulate signal, drive LEDs, monitor bateries (code available on my Github account)

PROJECT - Alarm Clock (Atmega)

This is one of my biggest project so far. It is going to be a digital digital clock showing seconds (if you want to). I'm using the Time library by Michael Margolis. I've not soldered this thing yet but I tested it on a breadbord and it seems to be working pretty well so far; the code is not completly finished yet as it only shows the current time for now but I'm gonna implement the other features in the near feature (well, I hope...). I wrote the code all by myself so it may not be perfetcly optimized... Let me know within the 'Contact me' section at the bottom of the page if you do have ideas for new features or things I could have done better

It does include:

  • Current time display (with seconds)
  • A date display mode
  • A buzzer (for the Alarm)
  • One led and a swith for the alarm
  • A potentiometer (to set the time / the alarm)
  • Multiples ringtones
  • An IR remote
  • Temperature (?) - yet to be done

Update 5/01/2016: I've been working a lot on this project lately and I soldered the "user" part on a prototype board. Also, the code is now in a final state! Only one bug left...
