Simon says game

This Simon game is my first soldered project. It is based on an Atmega 328.

Il does include:

  • An internal speaker
  • A switch to jack output (no sound if nothing connected)
  • A 7 segments display for score (best score at boottime)
  • A new game button (top right)
  • 4 green and red LEDs
  • 4 tactile buttons (if pressed after game over, last melody is played)
  • An acceleration feature

Capacitive sensor for lamp control (using Raspberry Pi)

This is my first project about home automation. It allows you to control one lamps through a relay (always be very carefull with high voltage ;)) just by touching a can. In this project, I'm using a Raspberry Pi but it can also be done with an Arduino or any microcontroller! I've added a touch feedback in the can with a small LED which turns off when you're touching the can. A Python script running as a deamon on the RPi does the logic.

I'm using a 1mâ„¦ resistor (one leg is connected to send pin and the other to receive pin). You then just have to conect the can to the receive pin. You may have to adjust the threshold value into the script

PROJECT - Alarm Clock (Atmega)

This is one of my biggest projects so far. It is going to be a digital digital clock showing seconds. I'm using the Time library by Michael Margolis. I've not soldered this thing yet but I tested it on a breadbord and it seems to be working pretty well so far; the code is not completly finished yet as it only shows the current time for now but I'm gonna implement the other features in the near feature (well, I hope...). I wrote the code all by myself so it may not be perfetcly optimized... Let me know within the 'Contact me' section at the bottom of the page if you do have ideas for new features or things I could have done better

Il does provide:

  • Current time display (with seconds)
  • A buzzer (for the Alarm)
  • A potentiometer (to set the time / the alarm)
  • Multiples ringtones
  • An IR remote
  • Temperature (?) - yet to be done

I'm going to post the code and schematics below when the project will be finished. But have a look at my Github account to if you want to see it or to check my advancement
